
MyShop – E-commerce Frontend Website
====================================

Project Description:
--------------------
MyShop is a fully responsive e-commerce frontend website developed using HTML, CSS, and JavaScript.
It simulates an online store showcasing products with functionalities like:
- Product listing grid
- Add to Cart button
- Price sorting (Low to High, High to Low)
- Category filtering UI (non-functional in current version)
- Dynamic cart total calculation

Folder Structure:
-----------------
/index.html           --> Main HTML file
/style.css            --> Embedded in index.html (inline for demo purposes)
/assets/images/       --> Product and banner images
/script.js            --> JavaScript logic (inline in HTML)

How to Run:
-----------
1. Download or clone the project files.
2. Open `index.html` in any modern web browser.
3. Browse the products, add them to the cart, and view real-time cart total.
